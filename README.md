# 概述

ABFrame是Application base framwork的缩写，


# 模块说明


1. abframe-config  统一的配置文件
2. abframe-entity  实体模型
3. abframe-util    工具类
4. abframe-manage  管理后台


# 安装

导入drill.sql，运行abrame-manage

登录： admin/1
